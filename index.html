SOP: Anggota pindah / keluar / masuk baru

Ikuti langkah ini supaya barcode selalu sesuai dengan anggota yang aktif.

1. Ada MUMI baru masuk

  1. Catat nama lengkap, kelompok, dan data dasar anggota baru di database utama (file Excel database MUMI).
  2. Kasih dia nomor ID baru (lanjutan nomor terakhir, jangan pakai ulang nomor bekas orang lain).
  3. Minta pengurus (Sekertaris/Bendahara) untuk update file aplikasi ini: tambahkan satu baris data di daftar anggota (ROSTER), lalu simpan ulang.
  4. Buka tab Cetak Barcode, cari nama anggota baru, unduh barcodenya, lalu kirim ke yang bersangkutan lewat WA.

2. Ada MUMI pindah kelompok

  1. Update kolom "Nama Kelompok" anggota tersebut di database dan di aplikasi (ID tetap sama, tidak perlu barcode baru).
  2. Kalau anggota ingin, boleh cetak ulang kartu barcode supaya label kelompoknya sesuai yang baru, ID QR-nya tetap sama seperti sebelumnya jadi riwayat kehadiran lama tidak hilang.

3. Ada MUMI keluar / pindah desa

  1. Jangan hapus datanya dari histori kehadiran lama (biar rekap lama tetap akurat).
  2. Tandai statusnya "non-aktif" di database utama, dan hapus baris orang tersebut dari daftar anggota (ROSTER) di aplikasi supaya tidak ikut dihitung di sesi berikutnya.
  3. Barcode lama otomatis tidak berlaku lagi begitu datanya dihapus dari aplikasi.

4. Barcode hilang / rusak / tertukar

  1. Tidak perlu ganti ID. Buka tab Cetak Barcode, cari namanya, unduh ulang, cetak lagi.
  2. ID lama tetap dipakai supaya rekap kehadiran sebelumnya tidak terputus.

5. Siapa yang boleh melakukan perubahan ini?

  • Perubahan data anggota (baru/pindah/keluar) dilakukan oleh Sekertaris, dikoordinasikan dengan Ketua/Wakil.
  • Kata sandi akses aplikasi ini hanya dibagikan ke pengurus, jangan disebar ke MUMI biasa.
